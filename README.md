# خزنده‌ی دیتابیس یونیورستالی (Universitaly Scraper)

این ابزار یک برنامه دسکتاپ با رابط کاربری مدرن (تم تاریک) است که به متقاضیان تحصیل و مهاجرت به ایتالیا کمک می‌کند تا به راحتی کل دوره‌های آموزشی دانشگاه‌های ایتالیا را جستجو، فیلتر و در قالب فایل اکسل (Excel) خروجی بگیرند. 

بسیاری از دانشجویان ایرانی برای مهاجرت تحصیلی به ایتالیا به دنبال دوره‌های **کارشناسی ارشد (Master's - Laurea Magistrale)** به **زبان انگلیسی (English)** و به صورت **حضوری (In Presence)** هستند. با استفاده از این نرم‌افزار، می‌توانید به جای ساعت‌ها جستجوی دستی در سایت شلوغ یونیورستالی، کل دوره‌های منطبق بر این شرایط را در کمتر از ۱ دقیقه استخراج کنید.

---

## ویژگی‌های کلیدی برنامه

* **رابط کاربری زیبا و مدرن (تم تاریک):** طراحی شده با کتابخانه استاندارد Tkinter پایتون و بهینه‌شده برای ویندوز.
* **سرعت فوق‌العاده بالا:** این ابزار داده‌ها را مستقیماً از API بک‌اند وبسایت یونیورستالی استخراج می‌کند (به جای استفاده از Selenium کند که نیاز به کلیک و بارگذاری مرورگر دارد) و فرآیند استخراج بیش از ۵,۸۰۰ دوره را در کمتر از ۲ دقیقه انجام می‌دهد.
* **فیلترهای پیشرفته و آماده:**
  * فیلتر پیش‌فرض تنظیم شده بر روی: **زبان انگلیسی**، مقطع **ارشد (۲ ساله)** و کلاس‌های **حضوری**.
  * امکان فیلتر بر اساس زبان (انگلیسی، ایتالیایی، همه).
  * امکان فیلتر بر اساس مقطع تحصیلی (لیسانس، ارشد، دکترا/پیوسته ۵ و ۶ ساله).
  * امکان فیلتر بر اساس نوع دسترسی (آزمون ورودی، بدون آزمون/آزاد، سهمیه‌ای).
  * امکان فیلتر بر اساس نحوه ارائه (حضوری، آنلاین، ترکیبی).
  * فیلتر بر اساس کلیدواژه در نام رشته (مثلاً Computer Science).
* **خروجی اکسل غنی و مرتب:** فایل خروجی شامل جزئیاتی چون نام دانشگاه، نام رشته به ایتالیایی و انگلیسی، مقطع، تعداد واحدها (CFU)، کد رشته، زبان، نحوه پذیرش، و لینک صفحه دوره است.
* **پردازش پس‌زمینه (Multi-threaded):** برنامه در هنگام استخراج قفل نمی‌کند و می‌توانید هر لحظه عملیات را متوقف (Stop) کنید.

## راهنمای نصب و راه‌اندازی برای کاربران ایرانی

برای اجرای این برنامه روی ویندوز، ابتدا باید پایتون را نصب داشته باشید.

### ۱. نصب پایتون (Python)
اگر پایتون را نصب ندارید:
1. به سایت رسمی [python.org](https://www.python.org/downloads/) بروید و آخرین نسخه ویندوز را دانلود کنید.
2. هنگام نصب، حتماً گزینه **Add Python to PATH** را تیک بزنید.

### ۲. نصب کتابخانه‌های پیش‌نیاز
ترمینال (PowerShell یا CMD) را باز کرده و دستور زیر را برای نصب کتابخانه‌های مورد نیاز اجرا کنید:

```bash
pip install requests pandas openpyxl
```

### ۳. اجرای برنامه
فایل‌های این پروژه را در یک پوشه قرار داده و در خط فرمان دستور زیر را وارد کنید تا برنامه باز شود:

```bash
python scraper_gui.py
```

---

## چطور از برنامه استفاده کنیم؟ (راهنمای گام به گام)

1. پس از اجرای دستور بالا، پنجره برنامه با فیلترهای پیش‌فرض مورد نیاز شما (رشته‌های ارشد انگلیسی حضوری) باز می‌شود.
2. در صورت تمایل می‌توانید فیلترها را تغییر دهید یا کلمه کلیدی خاصی بنویسید.
3. روی دکمه آبی‌رنگ **Scrape & Export to Excel** کلیک کنید.
4. یک پنجره برای شما باز می‌شود تا مسیر ذخیره فایل اکسل را مشخص کنید (مثلاً دسکتاپ را انتخاب کرده و نام فایل را `italy_courses.xlsx` بگذارید).
5. برنامه شروع به استخراج صفحات می‌کند. فرآیند پیشرفت کار در نوار پیشرفت و بخش لاگ‌ها به صورت زنده نمایش داده می‌شود.
6. در پایان، پیام اتمام کار به همراه تعداد دوره‌های استخراج شده نمایش داده می‌شود. حالا می‌توانید فایل اکسل را باز کرده و دوره‌ها را بر اساس دانشگاه یا کد رشته فیلتر کنید.

## ساختار ستون‌های خروجی اکسل
فایل خروجی شامل اطلاعات بسیار ارزشمندی برای اپلای است:
* **Course Name:** نام رشته به انگلیسی و ایتالیایی
* **University:** نام دانشگاه
* **Degree Type & Duration:** مقطع و مدت زمان دوره (مثلاً ارشد ۲ ساله)
* **CFU Credits:** تعداد واحدهای کل دوره (معمولاً ۱۲۰ واحد برای ارشد)
* **Subject Class Code:** کد کلاس رشته (مانند LM-18 برای علوم کامپیوتر یا LM-32 برای مهندسی کامپیوتر که برای ارزیابی سفارت بسیار مهم است)
* **Access Mode:** نوع پذیرش (آزاد یا نیاز به مدرک خاص)
* **Program Admission:** نوع پذیرش دانشگاهی (آزاد یا ظرفیت محدود)
* **URL:** لینک مستقیم به وبسایت دانشگاه برای بررسی شرایط پذیرش رشته
