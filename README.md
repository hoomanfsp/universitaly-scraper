# خزنده‌ی دیتابیس یونیورستالی (Universitaly Scraper)

این ابزار یک برنامه دسکتاپ با رابط کاربری مدرن (تم تاریک) است که به متقاضیان تحصیل و مهاجرت به ایتالیا کمک می‌کند تا به راحتی کل دوره‌های آموزشی دانشگاه‌های ایتالیا را جستجو، فیلتر و در قالب فایل اکسل (Excel) خروجی بگیرند. 

بسیاری از دانشجویان ایرانی برای مهاجرت تحصیلی به ایتالیا به دنبال دوره‌های **کارشناسی ارشد (Master's - Laurea Magistrale)** به **زبان انگلیسی (English)** و به صورت **حضوری (In Presence)** هستند. با استفاده از این نرم‌افزار، می‌توانید به جای ساعت‌ها جستجوی دستی در سایت شلوغ یونیورستالی، کل دوره‌های منطبق بر این شرایط را در کمتر از ۱ دقیقه استخراج کنید.

---

## 🚀 روش آسان: دانلود و اجرای مستقیم (بدون نیاز به نصب پایتون)

نیازی به نصب پایتون یا وارد کردن دستور در ترمینال نیست! ما برنامه را برای شما به صورت آماده کامپایل کرده‌ایم.

1. به صفحه **[انتشارها (Releases) در گیت‌هاب](https://github.com/hoomanfsp/universitaly-scraper/releases)** بروید.
2. آخرین نسخه را انتخاب کرده و فایل اجرایی متناسب با سیستم‌عامل خود را دانلود کنید:
   * **برای ویندوز:** فایل `UniversitalyScraper-Windows.exe` را دانلود کنید.
   * **برای لینوکس:** فایل `UniversitalyScraper-Linux` را دانلود کنید.
3. فایل دانلود شده را اجرا کنید تا پنجره برنامه باز شود! 
   *(در اولین اجرا روی ویندوز، ممکن است با اخطار Windows Defender SmartScreen مواجه شوید؛ در این حالت روی More Info و سپس Run Anyway کلیک کنید).*

---

## 🛠️ روش دوم: اجرا از طریق سورس پایتون (مخصوص توسعه‌دهندگان)

اگر می‌خواهید برنامه را از طریق کدهای پایتون اجرا کنید، مراحل زیر را دنبال کنید:

### ۱. نصب کتابخانه‌های پیش‌نیاز
ابتدا ترمینال (PowerShell یا CMD) را باز کرده و دستور زیر را اجرا کنید:

```bash
pip install -r requirements.txt
```

### ۲. اجرای برنامه
دستور زیر را وارد کنید تا برنامه باز شود:

```bash
python scraper_gui.py
```

## 🌟 راهنمای استفاده از نرم‌افزار

1. پس از باز شدن برنامه، فیلترهای پیش‌فرض مورد نیاز شما (رشته‌های ارشد، انگلیسی، حضوری و ۲ ساله) از پیش انتخاب شده‌اند.
2. در صورت تمایل می‌توانید فیلترها (زبان، مقطع، نوع پذیرش و...) را تغییر دهید یا کلمه کلیدی خاصی برای نام رشته وارد کنید.
3. روی دکمه آبی‌رنگ **Scrape & Export to Excel** کلیک کنید.
4. مسیر و نام فایل اکسل خروجی خود را انتخاب کنید (مثلاً در دسکتاپ ذخیره کنید).
5. برنامه شروع به استخراج صفحات می‌کند. فرآیند پیشرفت کار به صورت زنده نمایش داده می‌شود.
6. در پایان، فایل اکسل با جزئیات کامل ذخیره می‌شود و می‌توانید آن را در سیستم خود بررسی کنید.

## 📊 ستون‌های خروجی اکسل
فایل خروجی شامل اطلاعات کلیدی برای فرآیند اپلای و سفارت است:
* **Course Name:** نام رشته به انگلیسی و ایتالیایی
* **University:** نام دانشگاه
* **Degree Type & Duration:** مقطع و مدت زمان دوره (مثلاً ارشد ۲ ساله)
* **CFU Credits:** تعداد واحدهای کل دوره (معمولاً ۱۲۰ واحد برای ارشد)
* **Subject Class Code:** کد کلاس رشته (مانند LM-18 برای علوم کامپیوتر یا LM-32 برای مهندسی کامپیوتر که برای ارزیابی سفارت بسیار مهم است)
* **URL:** لینک مستقیم به وبسایت دانشگاه برای بررسی شرایط پذیرش رشته
